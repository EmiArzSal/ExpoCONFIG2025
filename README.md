# ExpoCONFIG2025

ExpoCONFIG2025 es un sistema para la gestión de exposiciones en ExpoESCOM. Permite registrar proyectos, asignar profesores y controlar visitantes.

## Tecnologías usadas
- 🌱 **Backend**: Spring Boot
- ⚛️ **Frontend**: React
- 🔥 **Base de datos**: Firebase Firestore
